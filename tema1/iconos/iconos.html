Funcionamiento iconos google

Para tener acceso a la api con los iconos tenemos que poner lo siguiente en la cabecera

link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons"

Para poner un icono en cuestion tendriamos que escreibir lo siguiente:

class="material-icons" style="color: yellow;"> bolt

Por ejemplo, un icono de una casa normal

house

Ahora probaremos a darle diferentes estilos, pondrmeos varios de diferentes tamaños y colores

school favorite loupe

Tambien podemos meter los iconos en botones

Funcionamiento iconos Bootstrap

Para usar los iconos que nos ofrece bootstrap, tendremos que poner el siguiente link en la cabecera
link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css"
y usaremos las clases glyphicon y glyphicon-icono en nuestro span para poner el icono, lo podemos meter donde queramos
